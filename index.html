Trabalho em Equipe com Controle e Segurança

Um Curso que explora as vantagens de ultilizar o Git como ferramenta de controle de versão para projetos em qualquer linguagem, em qualquer plataforma.

Principais beneficios:

  • funciona de maneira distribuida
  • Permita a edição concorrente de arquivos do projeto
  • Não depende de uma conexão ativa com um servidor
